# PageTag🔖
PageTag は、特定のURLルールに基づいてWebページにカスタムタグを自動表示するChrome拡張機能です。 
エンジニアやQA担当者が、複数の環境（Local, Staging, Production）を視覚的に一瞬で識別できるように設計されています。

## 🌟 主な機能
・正規表現によるフィルタリング: URLRegexpFilter により、特定のドメインやパスに対して柔軟にタグを表示。
・多彩なデザイン設定:
  スタイル: シンプル、ボーダー、斜線、格子、ハッチングの5種類。
  位置: 上下左右の4方向に配置可能。
  カラー: 16進数と透過度（Alpha）を自由にカスタマイズ。
・リアルタイム時計: タグ内に現在の時刻を表示可能。エビデンス用のスクリーンショット撮影時に便利です。
・インポート/エクスポート: 設定をJSONでバックアップ・共有可能。チーム間での設定共有も容易です。

## ⚠️ 注意事項 (Notice)
このプロジェクトは、Chromeウェブストアへの公開を目的としていません。 内部で jQuery を使用しているため、モダンなブラウザ拡張の制限に抵触する可能性があります。ご利用の際は、以下の「開発者モード」でのインストールをお願いします。

## 🛠 導入方法
1. このリポジトリを git clone または ZIPでダウンロード。
1. Chromeで chrome://extensions/ を開く。
1. 右上の「デベロッパー モード」をオンにする。
1. 「展開して読み込む」をクリックし、このフォルダを選択。

## 📖 使い方
1. 拡張機能のオプション画面を開きます。
1. 「New Marker」をクリックして新しい設定行を追加します。
1. 以下の項目を設定して「Save」を押します。：
   * Label: 表示したい文字
   * URL regexp filter: localhost や .*\.example\.com などの正規表現
   * そのほか: PositionやStyle・Colorなど
1. 該当するURLのページを開くと、自動的にタグが表示されます。
